# Personal Mixer Audio Files

This folder is served via GitHub Pages for hosting audio files.

## Usage

1. Place your MP3 file in the `audio/` folder (e.g., `example_01.mp3`)
2. Your file will be accessible at: `https://yourusername.github.io/repo-name/audio/example_01.mp3`
3. Update the `TRACK.url` in `backend/server.js` with this URL
